some_object:instance_of (some_class)
Checks whether some_object is instance of some_class. This function is dual to class_of.

Parameters:

  • some_class class to check against

Returns:

    true if some_object is instance of some_class; false otherwise

Parameters:

    true if some_class is class of some_object; false otherwise
some_class:init ([...])
Initializes an instance of some_class. This function is called when new instance of a class is created.

Parameters:

  • ... initialization parameters (optional)

Usage:

     local Foo = class()
 function Foo:init(bar)
   -- Bar.init(self, bar) -- call parent initializator if needed
   self.bar = bar
 end

 local foo = Foo:new('baz')
 print(foo.bar) -- prints 'baz'
some_class:new ([...])
Creates new instance of some_class. If defined, init function will be called to initialize the newly created instance.

Parameters:

  • ... parameters passed to the init function. (optional)

Returns:

    new instance of some_class

Usage:

     local Foo = class()
 function Foo:bar()
   print('baz')
 end

 local foo = foo:new()
 foo:bar() -- prints 'baz'
